Questões Teóricas

O que é a semântica em HTML e qual a sua importância no desenvolvimento de páginas web?

A semântica em HTML se refere ao uso de tags apropriadas que definem claramente o propósito e o significado do conteúdo. Por exemplo, o uso de <header>, <footer>, <article> e <section> torna o código mais legível e compreensível tanto para desenvolvedores quanto para mecanismos de busca, melhorando a acessibilidade e SEO.

Qual é a diferença entre uma classe e um ID em CSS e quando usar cada um?

Uma classe (.classe) pode ser aplicada a vários elementos e é usada quando você deseja aplicar o mesmo estilo a diferentes elementos. Já um ID (#id) deve ser único para cada elemento em uma página. Use classes para estilizar grupos de elementos e IDs para identificar um único elemento de forma específica.

O que é o conceito de 'box model' no CSS e como ele afeta o layout dos elementos em uma página?

O 'box model' descreve a estrutura de cada elemento HTML, que é composto por content (conteúdo), padding (espaçamento interno), border (borda) e margin (espaçamento externo). Ele afeta como os elementos são dispostos na página e como eles interagem entre si, especialmente em relação ao tamanho e posicionamento.

Como o Flexbox facilita a criação de layouts responsivos em comparação com métodos antigos, como float?

O Flexbox facilita o alinhamento e distribuição de elementos em contêineres flexíveis, permitindo uma melhor organização de layouts, tanto na vertical quanto na horizontal. Diferente do float, que tem limitações e requer "hacks", o Flexbox ajusta automaticamente os elementos de acordo com o tamanho da tela, tornando mais simples a criação de layouts responsivos.

O que é um reset CSS e por que é importante incluí-lo no início de uma página?

Um reset CSS remove os estilos padrão aplicados pelos navegadores, garantindo que todos os elementos comecem com as mesmas configurações. Isso evita inconsistências visuais entre diferentes navegadores e facilita a aplicação de estilos consistentes.
